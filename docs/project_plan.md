# Project Plan

## Project Name
Retail Analytics Engineering Platform

## Objective
Build a Snowflake and dbt analytics engineering project that transforms raw retail order data into tested, documented, analytics-ready models.

## Skills Demonstrated
- Snowflake cloud data warehousing
- dbt staging, intermediate and mart models
- Jinja and reusable macros
- Incremental models
- dbt tests and custom generic tests
- Data contracts
- Git and GitHub workflow
- GitHub Actions CI/CD
- Technical documentation
