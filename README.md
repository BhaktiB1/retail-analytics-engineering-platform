# Retail Analytics Engineering Platform

## Executive Summary
This project demonstrates an analytics engineering workflow for retail order data using Snowflake, dbt, automated testing, reusable macros, incremental models and GitHub Actions CI/CD.

The objective is to transform raw retail data into reliable, tested and documented analytics-ready models that could support internal BI and customer-facing analytics.

## Why This Project Exists
The project is designed to mirror a modern analytics engineering workflow:
- Load raw retail data into Snowflake
- Build dbt staging, intermediate and mart layers
- Apply automated data quality checks
- Use Jinja macros for reusable business logic
- Implement incremental materialisation for scalable models
- Add CI/CD using GitHub Actions
- Document the transformation layer clearly
